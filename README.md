## installation
### anaconda導入
https://www.anaconda.com/distribution/

### pytorch導入(仮想環境)
```
$ conda create -n pt python=3.5 -y
$ source activate pt
$ conda install -c pytorch pytorch=1.0.0 torchvision=0.2.2 -y
```
